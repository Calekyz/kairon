# KAIRON Trading System

Professional trading signal system for Deriv continuous indices.

## Quick Start

### Backend Setup
\`\`\`bash
cd backend
npm install
cp .env.example .env
# Edit .env with your credentials
npm run dev
\`\`\`

### Frontend Setup
\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`

## Database Setup
\`\`\`sql
CREATE DATABASE kairon_db;
\`\`\`

## Features
- One-device-at-a-time session protection
- Subscription management
- Real-time trading signals
- Professional dashboard
